## Success Criteria

1. ✅ `uvx mkdocs-dtinth --help` works
2. ✅ `uvx mkdocs-dtinth serve` serves docs with custom theme
3. ✅ `uvx mkdocs-dtinth build` builds docs with custom theme
4. ✅ User's `mkdocs.yml` is respected and merged
5. ✅ Minimal surface area (only serve/build commands)
6. ✅ Theme customizations are isolated and reusable
7. ✅ `mkdocs-dtinth init` creates working project skeleton
