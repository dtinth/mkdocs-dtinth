# mkdocs-dtinth

A reusable MkDocs distribution with dtinth's custom theme, designed for use across multiple projects via `uvx`.

## Quick Start

```bash
# Serve current directory as docs
uvx mkdocs-dtinth serve

# Initialize a new project
uvx mkdocs-dtinth init

# Build for production
uvx mkdocs-dtinth build
```
